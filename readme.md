# PHP缓存机制框架说明 #

## 项目依赖 ##
无。

## 使用说明 ##
 * 由静态常量CACHE_CLASS_NAME定义框架使用的缓存机制实现类。
 * 缓存机制实现类包括：
 	* SinaSAEMemcache：新浪SAE缓存机制实现类
